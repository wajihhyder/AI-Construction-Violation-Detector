# Frontend — AI-Powered Construction Violation Detection

React 18 + TypeScript + Vite single-page app for the Citizen, Authority, and Admin interfaces.

```bash
npm install
npm run dev      # http://localhost:5173 (proxies /api and /uploads to the backend on :8000)
npm run build    # production build to dist/
npm run lint     # ESLint
```

See the [root README](../README.md) for the full project overview, architecture, and setup.
